# LangGraph 学习资料整理

这个目录保存之前的 LangGraph 学习笔记和配套代码，和新的工程项目 `ai_agent_rag_system` 分开管理。

## 内容说明

- `01.串行.ipynb` 到 `08.人机交互.ipynb`：LangGraph 基础流程学习笔记。
- `langgraph实战.ipynb`：LangGraph 综合实战笔记。
- `langgraph流式输出.ipynb`：流式输出相关笔记。
- `工具调用.ipynb`：工具调用相关笔记。
- `数据流回述和修改.ipynb`：状态数据流转、回溯和修改相关笔记。
- `agent_excutor/`：Agent 执行器相关实验代码。
- `config_utils.py`：配置读取辅助代码。
- `model_factory.py`：模型创建辅助代码。
- `evn`：本地环境配置文件。

## 和新项目的关系

这些文件主要用于学习和实验。后续正式项目开发放在：

- `../ai_agent_rag_system/`

如果学习笔记里的代码需要复用到正式项目，建议先理解后再迁移，不要直接复制整个实验目录。
